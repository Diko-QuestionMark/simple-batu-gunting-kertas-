Game tentang batu gunting kertas<br>
Player dan musuh diberi 4 nyawa<br>
Jika player menang, maka darah musuh berkurang 1. Begitu juga sebaliknya<br>
Jika antara player atau musuh nyawanya bernilai 0. Maka game berakhir<br>
Baik player memenangkan game tersebut atau kalah, jumlah kalah menangnya dihitung dan ditampilkan di layar utama<br>
